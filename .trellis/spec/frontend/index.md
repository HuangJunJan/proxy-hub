## Pre-Development Checklist

Before frontend edits:

1. Read [Directory Structure](./directory-structure.md) before adding or moving files.
2. Read [Component Guidelines](./component-guidelines.md) before adding UI primitives or route forms.
3. Read [Quality Guidelines](./quality-guidelines.md) before touching setup, keys, theme, language, or tables.
4. Read [Type Safety](./type-safety.md) before changing API calls or DTOs.
